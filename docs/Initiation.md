## Database Initiation

再设置好开发环境后, 我们要初始化数据库. 这里我们选择直接用 SQL 文件在 PSQL 中初始化. 我们的初始化包含以下内容

* 数据库创建 + 管理员用户创建 - `ini.sql`
* 模式创建 + 索引创建 `schema.sql`
* 视图创建 `view.sql`
* 角色创建 + 授权 `auth.sql`
* 过程创建 `procedures.sql`
* 函数创建 `functions.sql`
* 触发器创建 `triggers.sql`
* 初始数据导入 `data.sql`
